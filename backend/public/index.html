users table

HTTP Methods Paths Descriptions
get /users/ retrieves all of the data in the table
get /users/:id retrieves all of the data in the table for a specific id
post /users/register adds a user to the database
patch /users/update/:id edits an existing user's data
delete /users/delete/:id delete an existing user

products table

HTTP Methods Paths Descriptions

get /products/:id retrieves all of the data in the table for a specific id
post /products/addProduct adds a product to the database
patch /products/update/:id edits an existing product's data
delete /products/delete/:id deletes an existing product
